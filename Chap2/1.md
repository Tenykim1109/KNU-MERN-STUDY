## npm 사용하기

> node package manager는 쉽게 말해, 다른 사람이 만들어 놓은 코드를 설치하고 사용할 수 있는 도구입니다

## npm 시작하기

1.  bash terminal에 입력

```js
npm init
```

- 프로젝트 정보를 담은 파일
